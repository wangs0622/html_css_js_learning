`sourceCode` 文件夹存放的是参考的源代码。

`step1` 文件夹是我迈出的第一步。编写了一个初始的静态网页，按照自己的理解，并参考源代码写的一个网页。觉得应该从静态的出发，然后在加入脚本语言，做成的动态的。
